Usage  

1.Setting env  
```  
export GIT_SITE = "https://xxx"  
export GIT_TOKEN = "xxx"  
export GIT_SERVER = "github"  
```

2.Execute crawl projects  
```
repofellow project remote owner
```

Development  
```
python3 -m repofellow project remote owner
```

Build  
```
python3 setup.py sdist bdist_wheel
```
